Telbee is an application that lets you listen to your online audiences & reply to them in your own voice. The Engin Tech Team created a Vox button for Telbee that you can use for your websites or branded pages you can link to from anywhere to allow your audiences to record voice messages for you directly through their browsers. This Vox plugin has two parts: hosted channel and plugin channel. Hosted Channel is Vox plugin which is hosted on Telbee’s own site and the user doesn’t have to integrate it or need any script for this to work. Plugin Channel is a wiglet that can be integrated into any website by simply copy-pasting the script into the header of their own site. Other than the Vox, we created a Channel Editor from scratch that allows customizations according to the user’s desire, while also choosing between the hosted or the plugin channel. Lastly all the user channels were then uploaded on Google Cloud asynchronously so that all the hits to the channels are server less. Using GraphQL all this time to communicate between the apps.
